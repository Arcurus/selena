# Selena - Self-Development Project

*Created: 2026-04-18*

## Purpose

A project for developing and optimizing this instance of Selena from the ground up. The goal is to create systems for research, planning, execution, reflection, and continuous self-improvement.

## Coordination Structure

**Three agents work together:**
1. **Parent Selena** (#heartbeats) - Main coordinator, oversight, approves big decisions
2. **Fast Heartbeat** (#selena-project, every 5min) - Operational support, checks, nudges
3. **Selena v2** (this project) - Working on Open World and self-development

**Communication:** Fast heartbeat reports to #selena-project, Parent monitors from #heartbeats.

## Core Ideas

- **Priority Reflection System**: Code that can evaluate what needs to be done and prioritize accordingly
